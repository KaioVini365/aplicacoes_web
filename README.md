# Caderno Financeiro

Aplicação web para controle de receitas e despesas com visual de caderno físico.

## O que faz

- Registra entradas e saídas com descrição, valor, data e categoria
- Calcula saldo em tempo real
- Lista o histórico de lançamentos
- Permite excluir registros individualmente
- Valida os campos antes de salvar

## Como usar

Abre o arquivo `index.html` direto no navegador. Não precisa de servidor.

## Tecnologias

- HTML5
- CSS3
- JavaScript puro (sem frameworks)

## Status

Projeto avaliativo — v1.0
